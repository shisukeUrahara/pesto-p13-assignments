# Book List React Application

## Overview

This React application displays a list of books, showcasing the use of React components, props, and CSS for styling. The project is designed with a dark theme and incorporates smooth and elegant UI/UX principles. It includes hover effects for enhanced user interaction and is structured with modularity in mind.

## Features

- **Displays a list of books** with titles, authors, and publication years.
- **Dark-themed user interface** for a visually appealing experience.
- **Hover effects** on book items for interactive feedback.
- **Modular CSS** for easy maintenance and scalability.

## Installation and Setup

Follow these steps to set up and run the project:

```bash
# Clone the repository
git clone [repository-url]

# Navigate to the project directory
cd my-book-list

# Install dependencies
npm install

# Run the application
npm start
The application will start running on localhost:3000.

Project Structure
src/App.js: The main application component.
src/BookList.js: Component to render the list of books.
src/Book.js: Component to render individual book details.
src/App.css: General styles for the application.
src/BookList.css: Styles specific to the BookList component.
src/Book.css: Styles specific to the Book component.
Technologies Used
React.js
HTML5
CSS3
```
